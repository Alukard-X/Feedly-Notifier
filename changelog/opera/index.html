Opera version changelog

From 2.6.17 version and upwards, Opera version uses the same codebase and release cycle as Chrome one. Please see changelog here.

v2.6.17 (2016.05.5)

  • Opera version was aligned with Chrome one.

v1.2.3 (2016.02.12)

  • Authentication URL was fixed.
  • Serbian translation was added.

v1.2.2 (2015.11.03)

  • Authentication URL was changed.
  • Icons for HiDPI displays were added.

v1.2.1 (2014.08.20)

  • Small font size in new Opera version was fixed.

v1.2.0 (2014.08.03)

  • Fixed re-login after closing the browser (Issue 1)
  • Added author to the full view (Issue 87)
  • Added option to open all feeds when popup is opened (Issue 83) (Issue 38)
  • Added option to open feeds in the same tab (Issue 82)
  • Added ability to set custom popup width. (Issue 40)
  • Added option to open Feeds in background tab.

v1.1.2 (2014.07.07)

  • Small bugfixes

v1.1.1 (2014.05.28)

  • Fixed authorization error (Issue 37)

v1.1.0 (2014.04.02)

  • Added 'Open all articles in 1 hit' feature. (Issue 74, Issue 34)
  • Added quick switch to enable / disable ability to save feed. Hold ctrl and click on the Feedly logo in the popup. (Issue 79)
  • UI improvements

v1.0.2 (2014.01.31)

  • Mustache.js was unminified

v1.0.1 (2014.01.31)

  • Translations with missed description were removed

v1.0.0 (2014.01.29)

  • Initial release
