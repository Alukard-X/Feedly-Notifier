Feedly Notifier

Google chrome extension for reading news from rss aggregator Feedly.

Feedly Notifier is a tiny extension that keeps you up to date with your feedly subscriptions.

Features

  • unread news count
  • unread news headers (with direct link on the news site) in the popup window
  • ability to mark news as read
  • ability to mark all news as read
  • article preview in the popup window
  • desktop notifications
  • background mode
  • ability save feeds
  • ability to set categories for update
  • ability to hide counter
  • ability to change font in the popup
  • HTTPS support
  • Added ability to filter feeds by category in the popup
